# Sankara Nigeria Limited - Agricultural Machinery Website

## About

Sankara Nigeria Limited is Nigeria's trusted partner for agricultural machinery. We specialize in premium tractors from Massey Ferguson, Lovol, and Zoomlion, along with farm implements, spare parts, and expert support.

## Project Overview

This is the official website for Sankara Nigeria Limited, built with modern web technologies to showcase our agricultural machinery products and services.

## Technologies

This project is built with:

- **Vite** - Fast build tool and development server
- **TypeScript** - Type-safe JavaScript
- **React** - UI library
- **shadcn-ui** - Component library
- **Tailwind CSS** - Utility-first CSS framework
- **React Router** - Client-side routing

## Getting Started

### Prerequisites

- Node.js (v18 or higher recommended)
- npm or bun

### Installation

1. Clone the repository:
```sh
git clone <YOUR_GIT_URL>
cd sankara-agri-grow
```

2. Install dependencies:
```sh
npm install
# or
bun install
```

3. Start the development server:
```sh
npm run dev
# or
bun run dev
```

The application will be available at `http://localhost:8080`

## Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run build:dev` - Build in development mode
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint
